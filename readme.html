Cómo usar el lector de códigos

Paso 1:

Descarga un Excel de tu catálogo desde Arcadia y pásalo a la PDA.

Paso 2:

Presiona el botón "Seleccionar archivo", busca dónde descargaste el Excel y selecciónalo. El lector te indicará si fue cargado correctamente.

Paso 3:

Ya está listo para usar. Escanea o escribe el producto.

Paso 4:

Después de encontrar un producto, el lector mostrará sus datos y debajo un botón de "Crear etiqueta", el cual te envía a Arcadia para generarla. Deberás iniciar sesión en Arcadia la primera vez. Puedes ir agregando los productos y luego crear todas las etiquetas juntas.

Información

La app está diseñada principalmente para escanear. Puedes escribir el producto, pero el proceso es tedioso.

Debajo del buscador se muestra el archivo que está cargado en la app; debes actualizarlo periódicamente.

Arcadia toma los precios de distintos lugares, por lo que algunos productos pueden no aparecer y no tenemos forma de acceder a ese archivo interno de Arcadia.
