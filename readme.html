Cómo usar el lector de códigos

Paso 1: Cargar el catálogo

Descarga el archivo Excel desde “Mi catálogo” en Arcadia y pásalo a la PDA. Podes hacerlo conectando la PDA por usb a la pc

Paso 2: Importar el archivo

Presiona el botón “Seleccionar archivo”, busca el Excel y selecciónalo. La app te avisará cuando el archivo se haya cargado correctamente.

Paso 3: Buscar o escanear productos

Una vez cargado el archivo, ya puedes usar el lector:
• Escanea el código de barras, o
• Escribe el nombre o código del producto

El sistema buscará automáticamente y mostrará la información.

Paso 4: Agregar productos al ticket

Cuando encuentres un producto, verás sus datos en pantalla. Debajo tendrás opciones para:

• Agregar (Sí) → Guarda el producto en una lista de tickets
• Cancelar (No) → Limpia la búsqueda actual

Puedes seguir agregando varios productos.

Paso 5: Descargar tickets

Cuando termines:

• Presiona “Descargar” para generar un archivo .txt con todos los productos
• Ese archivo se puede usar para imprimir etiquetas o como referencia

También puedes:
• Borrar tickets si quieres empezar de nuevo

Información importante

• La app está pensada principalmente para escanear; escribir productos es más lento.
• Debajo del buscador se muestra el archivo cargado actualmente. Es importante mantenerlo actualizado.
• Los precios dependen de Arcadia y pueden variar.
• Algunos productos pueden no aparecer, ya que Arcadia utiliza fuentes internas a las que no tenemos acceso.
